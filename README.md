# Notas : 

Este un programa de multiplicaciones con Node

```
Options:
      --help     Show help                                    [boolean]
      --version  Show version number                          [boolean]
  -b, --base     Es la base de la tabla de multiplicar
                                                    [number] [required]
  -l, --listar   Es la que muestra la tabla de multiplicar
                                             [boolean] [default: false]
  -h, --hasta    Es hasta donde quieres que llegue la tabla de
                 multiplicar                        [number] [required]
```
